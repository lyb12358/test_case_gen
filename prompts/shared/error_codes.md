### 3. TSP-APP错误码
| 错误码  | 说明                     |
|---------|--------------------------|
| 037000  | 参数不正确               |
| 037001  | 没有车辆使用权           |
| 037002  | 有其他指令正在执行       |
| 037003  | 已经存在该指令，不能重复执行 |
| 037004  | 指令下发失败             |
| 037005  | 唤醒失败(唤醒服务是否反馈) |
| 037006  | 请勿重复操作             |
| 037007  | 内部错误                 |
| 037008  | 需要刷新车况             |
| 037009  | 指令下发超时             |

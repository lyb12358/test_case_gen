#### 5.1.16. 远程授权启动、允许驾驶（ZAH）

##### 5.1.16.1 接口功能说明
本接口提供车主远程授权给当前用车人用车的功能。  
授权具体分为两种：允许远程启动；允许远程启动+远程解锁。

##### 5.1.16.2 接口使用说明
用户手持终端调用。

##### 5.1.16.3 接口设计说明

##### 5.1.16.4 接口协议与地址
```
POST /v1.0/remoteControl/control
```

##### 5.1.16.5 接口参数

| 参数名           | 类型                  | 必填 | 说明                                                                 |
|------------------|-----------------------|------|----------------------------------------------------------------------|
| serviceId        | String                | Y    | ZAH                                                                  |
| command          | String                | Y    | 指令类型：开: start                                                  |
| setting          | T                     | Y    | 其它参数                                                             |

**setting 内嵌参数：**

| 参数名               | 类型                     | 必填 | 说明                                                                                     |
|----------------------|--------------------------|------|------------------------------------------------------------------------------------------|
| serviceParameters    | List<ServiceParameter>   | Y    | 相关指令具体参数。一个 ServiceParameter 由一个 key 和一个 value 组成。                   |
| zah.pwd              | String                   | Y    | 使用 tcam 公钥加密后的密钥，证书接口参考 “4.1.5.4 获取车辆证书” 章节。                   |
| zah.rdu              | String                   | N    | 是否解锁：解锁="true"，闭锁="false"                                                      |

##### 5.1.16.6 接口输入样例
```json
{
  "serviceId": "ZAH",
  "command": "start",
  "setting": {
    "serviceParameters": [
      {
        "key": "zah.pwd",
        "value": "9765a1a09572b166877022963060c38c7dbcc63bb5b9ad34e2cd788a83099b2cc05ef3e449c0b45a3dd92dd01a22476ce48c509735a19528df5c8fd2be0481e37e3ac9e3c3ae42c66a9b5f6f94b5c5093642745512f03ef8b2d6ce813f2af845"
      },
      {
        "key": "zah.rdu",
        "value": "true"
      }
    ]
  }
}
```

##### 5.1.16.7 接口返回

| 参数名     | 类型   | 必填 | 说明     |
|------------|--------|------|----------|
| sessionid  | String | Y    | 会话ID   |

**示例：**
```json
{
  "respTime": 0,
  "code": "000000",
  "msg": "ok",
  "data": {
    "sessionId": "107-1658712640841"
  }
}
```

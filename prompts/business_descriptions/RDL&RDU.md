
#### 5.1.3. 锁定解锁车门 / 后备箱 / 引擎盖 / 加油盖 / 充电盖

##### 5.1.3.1 接口功能说明
本接口提供使用户通过 TSP 将应用程序中的命令发送到车辆上，以便远程锁定或解锁车门、后背箱（后背门）、引擎盖（前舱盖）。

##### 5.1.3.2 接口使用说明
用户手持终端调用

##### 5.1.3.3 接口设计说明

##### 5.1.3.4 接口协议与地址
```
POST /v1.0/remoteControl/control
```

##### 5.1.3.5 接口参数

| 参数名           | 类型                     | 必填 | 说明                                                                                                                                                                                                 |
|------------------|--------------------------|------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| vin              | String                   | Y    | 车辆唯一标识                                                                                                                                                                                         |
| serviceId        | String                   | Y    | 服务 ID：RDL: 锁定；RDU: 解锁                                                                                                                                                                        |
| command          | String                   | Y    | 指令类型: `start` 或 `stop`                                                                                                                                                                          |
| setting          | T                        | N    | 其它参数                                                                                                                                                                                             |

**setting 内部结构：**

| 参数名             | 类型                     | 必填 | 说明                                                                                                                                                                                                 |
|--------------------|--------------------------|------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| serviceParameters  | List<ServiceParameter>   | N    | 相关指令具体参数。一个 ServiceParameter 由一个 key 和一个 value 组成                                                                                                                                 |
| door               | String                   | N    | `front-left`: 主驾侧门；`front-right`: 副驾侧门；`back-left`: 左后门；`back-right`: 右后门；`all`: 所有门（目前 app 端不区分哪个位置的门，所以下发的是这个枚举值，tcam 端支持某个位置的门，但是无法唤醒模块）。多选使用多个 key 下发 |
| target             | String                   | N    | `trunk`: 后备箱（只支持解闭锁，开关还不支持；APP 端解锁支持，闭锁不支持）<br>`hood`: 引擎盖（纯电车没有）<br>`tank-flag`: 加油口盖（纯电车没有）<br>`front-charge-lid`: 前充电口盖（DC或AC/DC）<br>`back-charge-lid`: 后充电口盖（AC）<br>`private-lock`: 私人密锁（手套箱锁）<br>多选使用多个 key 下发<br>注：目前没有充电口盖解闭锁，只有打开关闭 |
| password           | String                   | N    | 私密锁密码，只允许4位数字                                                                                                                                                                            |
| time.start         | Int                      | N    | TCAM 打开门的开始时间，不填写表示立即开始。单位待定（只有解锁时才需要）；无需下发，tcam 目前没有用到该参数                                                                                           |
| operationScheduling| OperationScheduling      |      | 预约调度                                                                                                                                                                                             |
| duration           | Int                      | N    | （只有解锁时才需要）App 端传入的参数单位是分钟；无需下发，tcam 目前没有用到该参数                                                                                                                    |

##### 5.1.3.6 接口输入样例
```json
{
  "serviceId": "RDL",
  "command": "start",
  "setting": {
    "serviceParameters": [
      {
        "key": "door",
        "value": "all"
      }
    ]
  }
}
```

##### 5.1.3.7 接口返回

| 参数名     | 类型   | 必填 | 说明     |
|------------|--------|------|----------|
| sessionid  | String | Y    | 会话 ID  |

**示例：**
```json
{
  "respTime": 0,
  "code": "000000",
  "msg": "ok",
  "data": {
    "sessionId": "107-1658712640841"
  }
}
```

#### 5.1.2.6 远程空气净化

**接口功能说明**  
用户可以通过手机 APP 远程开启车内空气净化器，净化车内空气，每次开启 10（以车内配置为准）分钟后自动关闭，为驾驶员提供良好的驾驶/乘车环境。

**接口参数**  
车架号从请求头中获取

| 参数名     | 类型   | 必填 | 说明                             |
|------------|--------|------|----------------------------------|
| serviceId  | String | Y    | ZAM                              |
| command    | String | Y    | 指令类型：开: start 关闭: stop 必填 |

**接口输入样例**  
```json
{
  "serviceId": "ZAM",
  "command": "start"
}
```

**接口返回**

| 参数名     | 类型   | 必填 | 说明     |
|------------|--------|------|----------|
| sessionid  | String | Y    | 会话ID   |

**示例：**
```json
{
  "code": "000000",
  "msg": "ok",
  "data": {
    "sessionId": "1659679493045"
  }
}
```

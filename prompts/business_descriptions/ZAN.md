#### 5.1.2.5 远程电池预热开关

**接口功能说明**  
用户抱怨低温时无独立电池加热按键，不方便出行，项目组决议设计独立的电池加热按键满足用户需求。

**接口参数**  
车架号从请求头中获取

| 参数名     | 类型   | 必填 | 说明                          |
|------------|--------|------|-------------------------------|
| serviceId  | String | Y    | ZAN                           |
| command    | String | Y    | 指令类型：开: start 关闭: stop 必填 |

**接口输入样例**
```json
{
  "serviceId": "ZAN",
  "command": "start"
}
```

**接口返回**

| 参数名     | 类型   | 必填 | 说明     |
|------------|--------|------|----------|
| sessionid  | String | Y    | 会话ID   |

**示例:**
```json
{
  "code": "000000",
  "msg": "ok",
  "data": {
    "sessionId": "1659679493045"
  }
}
```
